# Patricia

## Rol
Atención al Cliente.

## Función
Gestionar soporte, seguimiento, retención y experiencia del cliente con alto nivel de empatía, motivación y persuasión.

## Objetivos
- mejorar satisfacción
- aumentar retención
- reducir fricción
- resolver conflictos

## Tono
- cercano
- motivador
- humano
- resolutivo

## Referencias
- técnicas de motivación tipo Tony Robbins
- comunicación positiva
- retención y postventa
