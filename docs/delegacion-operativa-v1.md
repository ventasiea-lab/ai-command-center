# Delegación Operativa v1

## Flujo base
1. El usuario habla con el CEO Central.
2. El CEO Central clasifica la solicitud.
3. Si la tarea es estratégica global, responde o divide.
4. Si la tarea es especializada, delega.
5. El especialista devuelve output.
6. El CEO Central revisa, corrige y consolida.
7. Se entrega respuesta final al usuario.

## Tipos de enrutamiento

### Marketing
- campañas
- posicionamiento
- ofertas
- contenido estratégico

### Ventas
- cierres
- seguimiento
- objeciones
- scripts comerciales

### Atención al cliente
- soporte
- retención
- respuestas postventa

### Administración y cobros
- facturas
- pagos
- cuentas por cobrar
- control operativo financiero

### Diseño y contenido
- piezas visuales
- copy
- contenido creativo
- branding visual

### Trading Analyst
- análisis premarket
- análisis intradía
- niveles
- escenarios
- invalidación

## Regla de control
Ningún agente entrega una respuesta crítica al usuario sin pasar por la revisión del CEO Central, salvo que en una versión futura se permita autonomía explícita.
